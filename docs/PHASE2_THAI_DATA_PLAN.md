# Phase 2: Thai Real Data Collection Playbook

## Goals
- Collect 5k+ ThinPrep images from 2-3 Thai hospitals
- IRB/PDPA compliant
- Multi-reader ground truth (2 pathologists)

## Steps
1. Partner hospitals (list in THAI_DATA_PLAYBOOK.md)
2. Annotation schema: Bethesda 5-class + koilocyte + quality
3. Active learning: Use current model to select uncertain cases
4. Timeline: 6-12 months to first 1000 images

## Ethics
- PDPA consent forms
- De-identification pipeline
- Hospital MOU template

See docs/THAI_DATA_PLAYBOOK.md for details.
