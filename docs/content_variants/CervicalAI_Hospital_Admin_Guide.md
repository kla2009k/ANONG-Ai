# CervicalAI — Hospital Admin & Lab Manager Implementation Guide (Community Hospital)

**Hardware Needed**: Existing microscope + USB camera or Foldscope adapter + Windows/Mac laptop (or Jetson) + stable power/internet optional

**IT Requirements**: Local network for demo, offline mode full, HIS/LIS stub via HL7 (Phase 2)

**Staff Training**: 2hr for lab tech (upload + review report), 1hr pathologist on XAI view + sign-off

**Cost**: Hardware ~25,000 THB one-time. Per screen marginal <5 THB. Maintenance: model update via signed package

**Integration**: LINE OA for patient (opt-in), dashboard for worklist, PDF/DOCX export

**Compliance Checklist**: PDPA consent, IRB if research, Thai FDA if claim SaMD, audit log review monthly

**Pilot Success Metrics**: Time-to-result <5min, override rate <25%, patient callback increase 20%+

**Risks & Mitigations**: Low quality images → quality guard; Drift → monitoring; Adoption → champion doctor
