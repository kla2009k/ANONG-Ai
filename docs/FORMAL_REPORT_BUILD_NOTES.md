## Purpose

This file is the formal Thai research-report version intended for serious
submission packaging. It is more formal than the complete booklet and follows a
school/research-report structure similar to the DermaTrace reference:

- cover page;
- inner cover page;
- approval/signature page;
- Thai abstract;
- English abstract;
- acknowledgements;
- manual table of contents;
- list of tables and figures;
- Chapter 1: introduction, research problem, objectives, scope, definitions;
- Chapter 2: related literature and clinical/AI framing;
- Chapter 3: methodology, dataset, architecture, metrics, web workflow, ethics;
- Chapter 4: results, confusion matrix, ROC, calibration, Grad-CAM, analysis;
- Chapter 5: conclusion, discussion, limitations, roadmap;
- references;
- appendices for intended use, risk register, judge Q&A, demo runbook,
  submission checklist, and protocol details.

The 2026-07-07 polished pass improved:

- Thai abstract and English abstract;
- Chapter 1 introduction, research problem, objectives, and research questions;
- formal bibliography with public-health, cytology, model, explainability,
  uncertainty, and medical-AI governance references;
- source-use table explaining what each reference group can and cannot support.

The report deliberately uses placeholders for author, advisor, school, and
committee names because the final submission identity was not provided in this
workspace.
