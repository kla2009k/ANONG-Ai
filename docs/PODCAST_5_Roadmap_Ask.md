# CervicalAI Roadmap & The Ask

## Phase 1 (DONE - POC)
Data pipeline (SIPaKMeD + RepoMedUNM + Mendeley)
EfficientNet-B0 + XAI (Grad-CAM + MC Dropout)
FastAPI + web demo
Two-layer reports (template)
Z-stack stub
All docs + pitch deck
Status: working demo, metrics on synthetic (real data needed for claims)

## Phase 2 (6-18 mo) - Differentiators
Thai fine-tuning (100-500 images, MOU + IRB)
Z-stack 2.5D (multi-channel or light 3D-CNN)
SAM2 editable demo (human-in-the-loop)
Edge deploy (ONNX on Jetson, <30s)
Local LLM patient report (optional, template default)
Claim: Thai-specific, z-stack, editable, edge-ready

## Phase 3 (18-36 mo) - Validation
Reader study (AI vs AI+MD vs MD)
Prospective validation (real workflow)
Cost-effectiveness (QALY)
Thai FDA SaMD awareness (not submission)
Pilot in 1-2 community hospitals (offline)
Claim: clinically validated, policy-ready

## Phase 4+ (Future)
WSI + MIL (full slides, not fields)
Federated + DP (privacy, multi-site)
Multimodal (image + HPV genotype + age)
Scale to CLMV

2. Mentorship: connect to NCI or regional pathology for data partnership
3. Recognition: this is socio-technical, not just algorithm
   - Loss-to-follow-up 41%
